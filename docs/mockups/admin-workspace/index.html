Stride Admin Workspace · prototype

Van lijst-en-loslaten
naar werkbank.

Statische klikbare mockups voor de eerste fase van de Admin Workspace-herinrichting — persoonslens. Geen backend, geen echte data: alles is hardgecodeerd om het interactiemodel te valideren. Vier oppervlakken, exact volgens de spec.

Oppervlak 01

Vandaag

De werklijst-thuispagina. Vijf wachtrijen met live tellingen, elk met de standaard bulkactie klaar. Het commandocentrum.

 Oppervlak 02

Inschrijvingen

De grid — het hart van de herinrichting. Multi-select, statusbewuste bulkbalk, groeperen, server-side paginering.

 Oppervlak 03

Dossier

Eén persoon, alle inschrijvingen. Volledige join: alle stages, offerte, aanwezigheid, keuzes, trajectvoortgang, en een geschiedenis-tijdlijn.

 Oppervlak 04

Trajecten

Meerdelige leertrajecten — verplichte cursussen plus keuzemodules. Lijst, detail met deelnemers, en doorklik naar de voorgefilterde grid.
Mockup-only · statische HTML + Alpine.js + CSS · geen build, geen API · opent als file://
